# Western poetics synthesis: the external frame

## The sequence

| Figure | Object of concern | The external frame that licenses/condemns it |
|---|---|---|
| Plato | Mimetic poetry generally | The pedagogical needs of the ideal city |
| Aristotle | Tragic content (violence, incest, kin-murder) | Formal plot structure (unity, probability, recognition) |
| Longinus | Excess, fault, transgression of proportion | The genius/greatness of soul of the author |
| Sidney | Any morally troubling content | The distinction between "right poets" / bad poets and readers |
| Neoclassicism | Onstage violence, breaches of decorum | Codified rules of *bienséance*, pre-existing the work |

## The throughline

Across five different answers to "what makes representation of transgressive material acceptable," none locates the answer *in* the representation itself. Each appeals to something standing outside the text — a civic project, a formal architecture, an author's stature, a class of proper readers, a rule system — and treats that external thing as though it were simply describing a fact about the text, when in fact it is *constituting* the categories ("harmful," "sublime," "corrupting," "decorous") under which the text gets judged.

This matters for the comparative project because it means the question "is *Jin Ping Mei* pornography or moral literature?" is the wrong question to import wholesale from a Western frame — not because Chinese criticism lacks an equivalent debate, but because the debate has the *same structure*: Zhang Zhupo's commentary is not discovering the novel's moral seriousness, it is constituting it, in exactly the way Aristotle's plot-structure or Longinus's genius constitutes the acceptability of the material it frames.

## Methodological upshot

Any comparative claim in this project should specify *which* external frame is doing the work in a given reading — civic-pedagogical (Plato-style), formal-structural (Aristotle-style), author-genius (Longinus-style), reader-class (Sidney-style), or codified-rule (neoclassical-style) — rather than treating "the Western tradition defends transgressive art" as a single undifferentiated position. The *Jin Ping Mei* commentarial tradition draws on more than one of these frames simultaneously (author-genius claims about the implied author; reader-class claims about 知音 vs. 俗人; formal-structural claims about retributive plotting), which is itself worth flagging as a point of difference from any single Western figure taken alone.
