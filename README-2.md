# Classics 124: Poetics — Comparative Frameworks Repository

A research repository tracing a throughline argument connecting defenses of transgressive representation in Western poetics to the commentarial tradition surrounding *Jin Ping Mei* (金瓶梅) and Judith Butler's theory of performativity.

## Overview

This repository stages a comparative argument across fourteen files, drawing together:

- **Western poetics**: Plato, Aristotle, Longinus, and Sidney on the defense (and critique) of transgressive or mimetic representation
- **Ming-Qing commentarial tradition**: Zhang Zhupo's 1695 commentary on *Jin Ping Mei* and the earlier Chongzhen commentary
- **Contemporary theory**: Judith Butler's performativity framework and Naifei Ding's *Obscene Things* (Duke UP, 2002)

## Key Textual Finding

A central finding developed here concerns the textual basis of David Roy's Princeton translation of *Jin Ping Mei*: Roy's translation draws on **Recension A** (*cihua*, 1610), while Zhang Zhupo and the Chongzhen commentator worked from **Recension B** (*xiuxiang*). This means Roy's English edition imports Zhang's interpretive frame onto a textual base it was not originally written to interpret — a mismatch with consequences for how English-language readers encounter the novel's commentarial tradition.

## Method

This project treats classical poetics and Ming-Qing fiction commentary as parallel — not merely analogous — traditions of theorizing transgressive representation, using Butler's performativity as a connecting hinge rather than an anachronistic overlay.
